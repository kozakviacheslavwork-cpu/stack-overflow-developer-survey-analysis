# Stack_Overflow_Analysis_Project
# Аналіз результатів опитування Stack Overflow (Python)

## Про проєкт
Це підсумковий проєкт курсу Python, де я аналізую дані щорічного опитування розробників Stack Overflow. Мета — дослідити ринок праці, рівень зарплат та популярність мов програмування серед професіоналів.

## Що було зроблено:
* **Аналіз великого датасету:** Робота з даними понад 49,000 респондентів.
* **Статистика:** Обчислено середній, медіанний та найчастіший (мода) досвід роботи розробників.
* **Фільтрація та сегментація:** - Розраховано відсоток користувачів Python.
  - Проаналізовано рівень зарплат (75-й перцентиль) серед віддалених працівників.
  - Виявлено найпопулярніші індустрії серед високооплачуваних спеціалістів.
* **Освіта:** Досліджено зв'язок між рівнем освіти та рівнем доходу.

## Технології та методи:
* **Pandas:** Groupby, Aggregation, Pivot tables, Quantile analysis.
* **Data Cleaning:** Робота з пропущеними значеннями (NaN) та множинними відповідями у рядках.
